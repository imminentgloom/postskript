# postskript

postskript is a sampler for arc and norns.  
postskript is fast, momentary and volatile.  
no files can be loaded, nothing is saved.
  
you hold to record, then turn the dials. that's it.

```
controls, arc:
E1 - rate
E2 - level
E3 - start
E4 - length
K1 - hold to record
```
```
controls, norns:
E1 - rate
E1 + K1 - level
E2 - start
E3 - length
K2 - hold to record
K3 - clear
```

_arc recomended. complemented by a grid, sidvagn or gridkeys, and ii via nb._
